# **App Name**: Digital Morph

## Core Features:

- Dynamic Content Management: Enable content editing and management for all website sections via Firebase.
- Team Hierarchy Display: Showcase the team's structure visually and interactively with expandable details for each member, sourced dynamically from Firestore.
- Services Showcase: Present available services in a grid of morphed cards, each providing a brief description, an icon, and a 'Learn More' option, driven by data stored in Firestore.
- Interactive Contact Form: Implement a contact form with Firebase integration to save submissions and trigger optional push notifications, using a tool for enhanced delivery confirmation.
- Real-time Content Updates: Utilize Firestore Studio integration for immediate content updates across all sections, enhancing responsiveness and data accuracy.
- Enhanced User Interaction: Incorporate smooth morph/neumorphism shadows for buttons and cards and interactive expandable team cards to capture users attention and simplify the information for end-users.
- Dark/Light Mode Switching: Enable users to switch between light and dark themes.

## Style Guidelines:

- Background color: #ebf2fa, providing a light and airy feel.
- Primary color: #05668d, a strong, professional blue to reinforce brand identity.
- Accent color: #427aa1, complementing the primary with a slightly lighter shade for highlights and calls to action.
- Font: 'Inter', a clean, modern sans-serif font for readability and a professional appearance. Note: currently only Google Fonts are supported.
- Use a fully responsive layout ensuring adaptability across desktop, tablet, and mobile devices.
- Employ animated SVG icons that reflect each service category, enhancing visual engagement.
- Implement smooth morph/neumorphism shadows for interactive elements, with soft transitions and animated gradients in the hero section.
